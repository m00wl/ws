# `ws` – my personal website

This repository contains my personal website.

## Design Decisions

The goal of this project is to create a fast, lightweight and portable website that is still visually pleasing, easily extendable and deployable.
Those requirements lead to some interesting design decisions.
Be advised that these are strongly opinionated and may not reflect your views/suit your needs!

The website should be:
- deterministically generated by a static site generator (i.e. [`jekyll`](https://jekyllrb.com/))
- [designed to last](https://jeffhuang.com/designed_to_last/):
  - single page
  - no `javascript`
  - minimal, ideally classless, `CSS` definitions
  - `<100kB` weight in total
- optimized for accessability

## Repository Structure

The structure of this repository is based on the typical `jekyll` skeleton.
It is designed with a hybrid approach in mind.
This means that the repository contains both a `flake.nix` to build and host the website locally with `nix` as well as a `Gemfile` for non-nix builds with `bundler` and `jekyll` (e.g. for hosting on GitHub).

## Usage

- nix:
```shell
nix build
nix run
```
- non-nix:
```shell
bundle exec jekyll build
bundle exec jekyll serve
```
